## CSS selectors I used (at least 5)
- **Element selector:** `body`, `h1`, `img` (to set global base styles).
- **Group selector:** `h1, h2, h3` (to apply consistent typography across all headings).
- **Descendant selector:** `nav ul`, `nav li`, `nav a` (to specifically style the navigation structure).
- **Class selector:** `.card` (used for modular content blocks).
- **ID selector:** `#works` (to give a unique accent color to the portfolio section).
- **Pseudo-class:** `a:hover` (to create interactive neon lighting effects).
