# JavaScript Features

## Features Implemented
- **Theme Toggle (Feature A):** Switches the entire page between a dark Cyberpunk mode and a clean light mode by toggling CSS variables.
- **Click Counter (Feature B):** Tracks user "Energy Pulses" and logs the total interaction count to the browser console.

## How to Test
- **Toggle Theme:** Click the button labeled **"Switch Theme"** (`#themeBtn`). You will see the background and text colors transition.
- **Click Counter:** Click the button labeled **"Energy Pulse"** (`#clickBtn`), then open your browser console (**F12**) to view the live click count and system logs.
